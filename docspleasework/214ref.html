Java source files should always be named with the name of the class they contain (case-sensitive), followed by the .java suffix. By convention, Java class names begin with an uppercase letter and use camel case.

To compile: javac -deprecation SourceFileName.java
To run: java SourceFileName

SINGLE LINE:

There are 3 print methods: print, println, and printf. The + operator can be used to concatenate strings, as shown in the following example:


      System.out.println("\nThe area is " + circleArea(radius) + "\n");


When a real number is printed using printf, you can control the number of decimal places being displayed as shown in this statement which prints the radius to 15 decimal places:

      System.out.printf("The area is %.15f\n\n", circleArea(radius));


Sequence for formatting strings is % [flags] [width] [.precision] conversion-character
Conversion characters:
d : decimal integer [byte, short, int, long]
f : floating-point number [float, double]
c : character Capital C will uppercase the letter
s : String Capital S will uppercase all the letters in the string
h : hashcode A hashcode is like an address. This is useful for printing a reference
n : newline Platform specific newline character- use %n instead of \n for greater compatibility

Files should end in .clj and be saved in the clojure/src folder.

To compile and run -main function: clojure -m file_name

Header comments should begin with 4 semicolons


    	;;;; circle_area.clj calculates the area of a circle.
    	;;;;
    	;;;; Input: The radius of a circle.
    	;;;; Output: The area of that circle.
    	;;;;
    	;;;; Usage: clojure -m circle_area
    	;;;;
    	;;;; Begun by: Prof. Adams, for CS 214 at Calvin College.
    	;;;; Completed by:
    	;;;; Date:
    	;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;

Important comments (e.g., function description) should begin with 3 semicolons.

    	;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;
    	;;; Function -main is where execution begins
    	;;; Input: the radius of a circle.
    	;;; Output: the area of that circle.
    	;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;
Comments that clarify a section of code within a function should begin with 2 semicolons. Comments about a single line of code should begin with 1 semicolon.

Use puts to add a newline after executing:

      puts circleArea(radius)

basic input:
name = gets

Because gets returns a string, it can be used with string methods, for example:
gets.chomp -- chomp is a string method that removes trailing \n character
gets.to_i -- converts input to integer
gets.to_f converts input to float
These methods can also be used in conjunction with one another, such as in gets.chomp.to_i.
